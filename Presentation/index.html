FolioLLM

Constructing Portfolio of ETFs using Large Language Models

Stanford CS224N Custom Project

Andrey Popov, Oleg Roshka, Ilia Badanin

Introduction

Our goal is to develop FolioLLM, a domain-specific large language model designed to assist investors and financial professionals in constructing portfolios of exchange-traded funds (ETFs).

We aimed to fine-tune an LLM to understand user preferences, qualitative and quantitative investment features of funds, and portfolio optimization techniques, providing personalized investment advice.

Problem & Data Extraction

Extracted and prepared a comprehensive dataset from financial data providers such as Bloomberg and Refinitiv.

Gathered data on 12,224 ETFs, including both qualitative and quantitative attributes.

Collected price history for these ETFs to compute financial metrics and solve optimization problems.

Markowitz Optimization

Utilized Markowitz's mean-variance optimization framework to construct efficient portfolios.

Maximized expected returns for a given level of risk.

Model Fine-Tuning

Evaluated various pre-trained language models, including BERT, Zephyr-7b, GPT-2, and Alpaca.

Fine-tuned the FinGU model with 0.5B parameters using synthetic and real ETF data.

Training Pipeline

Classical Fine-Tuning: Updated all model parameters to fit our ETF-specific task.

LoRA: Injected trainable low-rank matrices for efficient fine-tuning.

Modified LoRA with KAN: Replaced one LoRA matrix with a KAN layer to better extract nuanced relationships.

Inference Process

Utilized Retrieval-Augmented Generation (RAG) to enhance context for generating responses.

Employed classification techniques to determine if a query required portfolio optimization.

Results

Significant improvements in model performance compared to base models.

Higher BERT Recall, lower Perplexity, and higher Cosine Similarity.

Superior financial metrics in response portfolios with higher average returns and lower volatilities.

Conclusion

FolioLLM demonstrated potential in the financial domain for automating and personalizing portfolio construction.

Fine-tuning and innovative techniques like KAN and RAG yielded promising results.

Future improvements with larger datasets and further exploration of fine-tuning approaches.
